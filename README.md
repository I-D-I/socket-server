# Socket-server

## Comandos
```
npm init (damos nombre socker-server)

npm i express
npm i cors
npm i body-parser

npm install typescript -g
tsc --init

````

## generar el DIST
```
tsc -w
````

# levantar el servidor
```
nodemon dist/
````

## algunos paquetes adicionales para dev
```
npm i --save-dev @types/express
npm i --save-dev @types/cors
```
